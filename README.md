# CTS-CSharp-Solution

This repository contains a simple C# console program that prints the sum of two numbers.

## Code Details
- Language: C#
- File: Program.cs
- Output: Sum = 30

## How to Run (two common ways)

### Using .NET SDK (recommended)
1. Ensure .NET SDK is installed (https://dotnet.microsoft.com/download).
2. From project folder: dotnet run

### Using Mono (older method)
1. Compile: mcs Program.cs
2. Run: mono Program.exe
